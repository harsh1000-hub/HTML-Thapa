Text Formatting Tag

For text Formatting firstly we use bold tag <b>

Bold Text

To define the important text we use strong tag <strong>

My name is Harsh

To define text style as Italic we use Italic tag <i>

Hi very good morning

To emphasize the text we use emphasize tag <em>

This is emphasize text

To marked the text (background color) we use mark tag <mark>

Hey this is mark tag

To make the text smaller we use small tag <small>

Hey we use small tag

To struck/make a line on text we use del tag <del>

Oops something went wrong

To write the text as subscript means like chemical formula we use subcript tag <sub>

H2SO4

To write the text as superscript means like math formula we use subcript tag <sub>

a2+b2

To write text from right to left we use bdo tag <bdo>

HARSH VARDHAN

To write text from left to right we use bdo tag <bdo>

HARSH
